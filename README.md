# java-stack-array-implementation
A simple Stack Data Structure implementation in Java using arrays. Supports push, pop, and display operations with stack overflow and underflow handling. Ideal for beginners learning Data Structures and Algorithms (DSA) concepts in Java.
#Stack Using Array in Java

A simple implementation of the Stack Data Structure in Java using arrays.

## Features

- Push operation
- Pop operation
- Display stack elements
- Stack Overflow handling
- Stack Underflow handling
